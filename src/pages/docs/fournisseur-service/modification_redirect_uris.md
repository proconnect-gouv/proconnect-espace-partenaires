# ⚙️ Modification des redirect_uri

Si vous souhaitez changer les redirect_uri de connexion ou de déconnexion configurées par ProConnect, voici la procédure.

## 1. Mon application est sur Internet

Pour l'intégration, cela se passera directement sur [votre espace](../../apps). Pour un changement en production, vous pourrez faire la demande en deux minutes sur [ce lien](https://www.demarches-simplifiees.fr/commencer/demande-de-modification-d-un-fournisseur-de-service).


## 2. Mon application est sur le RIE

Pour un changement en intégration ou en production, vous pourrez faire la demande en deux minutes sur [ce lien](https://www.demarches-simplifiees.fr/commencer/demande-de-modification-d-un-fournisseur-de-service).
